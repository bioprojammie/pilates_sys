🧘‍♀️ Pilates Pro - 個人皮拉提斯教學管理系統
這是一個專為皮拉提斯（Pilates）老師設計的輕量級、全功能管理系統。無需下載 App，直接透過瀏覽器即可在 iPad 或電腦上管理學生資料、課程排程、電子合約與課綱紀錄。

🔗 立即使用 (Live Demo)
點擊下方連結即可開啟系統：
👉 https://bioprojammie.github.io/pilates_sys/

✨ 主要功能
📱 iPad 最佳化體驗：支援「加入主畫面」全螢幕模式，操作流暢如原生 App。

✍️ 雙向電子簽名：支援學生與老師雙方簽名，適用於入會合約、購買堂數與上課簽到。

📅 週曆總表：自動彙整所有學生的固定上課時間，一週課表一目瞭然。

📸 課綱圖解標註：可上傳動作照片，並直接在照片上畫線、圈選重點（支援紅/黃/白筆刷）。

📝 預排與簽到：可提前規劃課程內容（預排），上課當天再給學生簽名扣課。

💾 自動儲存：資料儲存於裝置瀏覽器中（Local Storage），無需連網資料庫。

📲 如何在 iPad 上安裝 (推薦)
為了獲得最佳的全螢幕體驗（隱藏網址列），強烈建議將此網頁加入 iPad 主畫面：

使用 Safari 瀏覽器 開啟 系統網址。

點擊右上角的 「分享」 按鈕（方塊上有一個向上箭頭的圖示）。

往下滑動選單，點選 「加入主畫面」(Add to Home Screen)。

輸入名稱（例如：Pilates Pro），按下 「新增」。

回到桌面，點擊新出現的 App 圖示即可全螢幕使用。

📖 使用教學 (User Guide)
1. 新生建檔 (Onboarding)
點擊下方導航列的 「建檔」。

輸入學生姓名、購買堂數、身體狀況備註。

設定 「預設固定上課時間」（例如：週一 10:00），可加入多個時段。

合約管理：

點擊按鈕上傳紙本合約的照片或截圖。

在下方簽名板完成 學生 與 老師 的雙向電子簽名。

按「完成建檔」。

2. 總表與週曆 (Dashboard)
點擊 「總表」 頁籤。

上方 週曆 會顯示本週所有排定的課程，點擊卡片可直接進入該位學生頁面。

下方列表顯示所有學生及其剩餘堂數。

3. 課程規劃與簽到 (Class Management)
進入學生個人頁面後：

預排課綱 (課前準備)：

點擊相機圖示上傳動作照片，或輸入文字備註。

點擊照片可進入 「畫圖模式」，在照片上標註重點。

按下 「暫存為預排」 (此時不會扣堂數)。

上課簽到 (課後確認)：

點擊歷史紀錄中黃色的「預排」項目。

請學生在下方的簽名板簽名。

按下 「確認扣課」，系統將自動扣除 1 堂課，並壓上時間戳記。

4. 續約與購課 (Renew)
在學生個人頁面右上角，點擊 「購課/續約」 按鈕。

輸入購買堂數。

完成 學生 與 老師 雙方簽名確認。

按下確認後，堂數會自動累加，並在歷史紀錄留存一筆購買紀錄。

⚠️ 資料安全提醒
本系統採用 單機網頁技術 (Local Storage)，所有資料（學生檔案、照片、簽名）皆儲存在您目前使用的 這台裝置的瀏覽器內。

請勿 清除瀏覽器的「快取資料」或「網站資料」，否則資料會消失。

如果您更換 iPad 或使用不同手機，資料不會自動同步（因為沒有後端伺服器）。

建議定期截圖備份重要的合約畫面。
