## ✨ 核心功能特色

### 📱 **完美跨裝置體驗**
推薦使用 iPad 並「加入主畫面」，享受如同原生 App 般的全螢幕流暢操作。

### ✍️ **雙向電子合約**
* 內建合約上傳功能。
* 支援 **「學生」** 與 **「老師」** 雙方電子簽名，保障教學權益。

### 📅 **視覺化週曆**
自動彙整所有學生的「固定上課時間」，每週行程一目瞭然，不再怕衝堂。

### 📝 **卡片式課程管理 (核心功能)**
* **預排課程**：提前規劃課程內容與筆記。
* **多圖紀錄**：支援一次上傳多張照片，並可直接在照片上 **畫線標註重點**。
* **簽到扣課**：上課當日學生簽名確認後，系統自動扣除堂數並鎖定紀錄。

### 🛡️ **資料管理與封存**
* **封存功能**：可將暫停上課的學生「封存」，保持總表乾淨，且隨時可一鍵還原。
* **自主備份**：內建一鍵備份與還原功能，換手機也能輕鬆轉移資料。

## 📲 安裝教學 (iOS/iPadOS 推薦)

為了獲得最佳的全螢幕體驗（隱藏瀏覽器網址列），請依照以下步驟操作：

1. 使用 **Safari 瀏覽器** 開啟 [系統網址](https://bioprojammie.github.io/1on1_Pro/)。
2. 點擊右上角的 **「分享」** 按鈕（方塊上有一個向上箭頭的圖示 ⬆️）。
3. 往下滑動選單，點選 **「加入主畫面」(Add to Home Screen)**。
4. 輸入名稱（例如：1on1 Pro），按下 **「新增」**。
5. 回到桌面，點擊新出現的 App 圖示即可全螢幕使用。

## ⚠️ 資料安全與備份 (非常重要)

本系統採用 **單機網頁技術 (Local Storage)**，所有資料皆儲存在您目前的裝置瀏覽器中。

> **我們不會將您的資料上傳至任何雲端伺服器，這意味著只有您能看到這些資料，保障絕對隱私。**

請務必注意以下事項：

* ❌ **請勿清除瀏覽器快取 (Cache)**：這會導致資料遺失。
* 💾 **定期備份**：請定期至「設定」頁面點擊 **「下載備份檔案」**，並將檔案妥善保存（建議存於 iCloud Drive 或 Google Drive）。
* 📱 **更換裝置**：若需更換手機或平板，請先在舊機執行備份，再於新機執行「還原資料」。
