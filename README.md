🎓 1on1 Pro - 個人教學管理系統
這是一個專為自由接案老師（健身教練、家教、樂器老師、治療師等）設計的輕量級教學管理系統。無需下載 App，直接透過瀏覽器即可在 iPad 或手機上管理學生資料、課程排程、電子合約與課綱紀錄。

🔗 立即使用 (Live Demo)
點擊下方連結即可開啟系統：

👉 https://bioprojammie.github.io/1on1_Pro/
✨ 核心特色
📱 跨裝置支援：推薦使用 iPad 並「加入主畫面」，享受全螢幕 App 般的操作體驗。

✍️ 雙向電子合約：內建合約上傳與「學生/老師」雙方電子簽名功能，保障教學雙方權益。

📅 視覺化週曆：自動彙整學生的固定上課時間，每週行程一目瞭然。

📝 卡片式課程管理：

預排功能：可提前規劃課程內容與筆記。

多圖紀錄：支援一次上傳多張照片，並可直接在照片上畫線標註重點。

簽到扣課：上課當日學生簽名確認後，系統自動扣除堂數並鎖定紀錄。

💾 資料自主備份：內建一鍵備份與還原功能，換手機也能輕鬆轉移資料。

🗄️ 封存管理：可將暫停上課的學生封存，保持版面整潔，隨時可還原。

📲 安裝與使用 (iOS/iPadOS 推薦)
為了獲得最佳的全螢幕體驗（隱藏網址列），請務必執行以下步驟：

使用 Safari 瀏覽器 開啟 系統網址。

點擊右上角的 「分享」 按鈕（方塊上有一個向上箭頭的圖示）。

往下滑動選單，點選 「加入主畫面」(Add to Home Screen)。

輸入名稱（例如：1on1 Pro），按下 「新增」。

回到桌面，點擊新出現的 App 圖示即可全螢幕使用。

⚠️ 資料安全與備份 (重要)
本系統採用 單機網頁技術 (Local Storage)，所有資料皆儲存在您目前的裝置瀏覽器中，不會上傳至任何雲端伺服器，這意味著只有您能看到這些資料。

請勿清除瀏覽器快取：這會導致資料遺失。

定期備份：請定期至「設定」頁面點擊 「下載備份檔案」，並將檔案妥善保存（如存於 iCloud Drive）。

更換裝置：若需更換手機或平板，請先在舊機執行備份，再於新機執行「還原資料」。
