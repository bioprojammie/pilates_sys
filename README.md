🧘‍♀️ Pilates Pro - 皮拉提斯教學管理系統 (V8.0)
這是一個專為皮拉提斯（Pilates）老師設計的輕量級、全功能管理系統。無需下載 App，直接透過瀏覽器即可在 iPad 或電腦上管理學生資料、課程排程、電子合約與課綱紀錄。

🔗 立即使用 (Live Demo)
點擊下方連結即可開啟系統：

👉 https://bioprojammie.github.io/pilates_sys/
✨ 核心特色
📱 iPad 最佳化體驗：支援「加入主畫面」全螢幕模式，操作流暢如原生 App。

✍️ 雙向電子簽名：支援學生與老師雙方簽名，適用於入會合約、購買堂數與上課簽到。

📅 週曆總表：自動彙整所有學生的固定上課時間，一週課表一目瞭然。

📸 課綱圖解標註：可上傳動作照片，並直接在照片上畫線、圈選重點（支援紅/黃/白筆刷）。

💾 資料備份與搬家：內建一鍵備份功能，換平板也能輕鬆轉移資料。

📲 如何在 iPad 上安裝 (強烈推薦)
為了獲得最佳的全螢幕體驗（隱藏網址列），請務必執行以下步驟：

使用 Safari 瀏覽器 開啟 系統網址。

點擊右上角的 「分享」 按鈕（方塊上有一個向上箭頭的圖示）。

往下滑動選單，點選 「加入主畫面」(Add to Home Screen)。

輸入名稱（例如：Pilates Pro），按下 「新增」。

回到桌面，點擊新出現的 App 圖示即可全螢幕使用。

📂 資料備份與還原 (重要功能)
由於本系統是單機版，更換裝置前請務必進行備份。

1. 如何備份 (匯出資料)
在 「總表」 頁面的最下方，找到灰色區域。

點擊 「備份資料 (下載)」 按鈕。

系統會產生一個 .json 檔案（檔名會有日期），請將此檔案儲存到您的 iCloud Drive 或傳送到新裝置。

2. 如何還原 (匯入資料)
在新裝置（或重灌後）打開本系統。

點擊下方的 「還原資料 (上傳)」 按鈕。

選擇您剛剛下載的 .json 備份檔。

系統會詢問確認，按下確定後，所有學生資料、照片、簽名就會全部回來了！

⚠️ 單機版注意事項 (請務必閱讀)
本系統採用 單機網頁技術 (Local Storage)，這意味著：

資料只存在這台機器裡：

您的資料不會自動上傳到雲端。

您在 iPad A 輸入的資料，不會出現在 iPad B。

如果您把 App 刪除，資料就會全部消失。

請勿清除瀏覽器快取：

請小心不要在設定中清除 Safari 的「網站資料」或「快取」，這會導致資料遺失。

定期備份：

強烈建議您每週或每月執行一次「備份資料」，並將檔案存到雲端硬碟（Google Drive / iCloud），以防平板遺失或損壞。

📖 使用教學 (User Guide)
新生建檔：輸入姓名、購買堂數，並完成「學生」與「老師」的合約雙向簽名。

課程規劃：可提前上傳動作照片並畫重點（預排），上課當天再給學生簽名並按「確認扣課」。

續約購課：在個人頁面點擊「購課/續約」，輸入堂數並完成雙方簽名，系統自動累加堂數並留存紀錄。
